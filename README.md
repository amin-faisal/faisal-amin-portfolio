# Faisal Amin® — Portfolio

Personal portfolio of Faisal Amin, Senior Product Designer.

Built with **React + Vite + Framer Motion + Lenis** — dark mode, Inter typography,
purple/black branding, blur-in section reveals, custom cursor, magnetic buttons
and page transitions.

## Pages

- `/` — landing
- `/work` — all shipped projects
- `/about` — story, experience timeline, education
- `/services` — services & process
- `/contact` — contact & next steps

## Development

```bash
npm install
npm run dev
```

## Deployment

Pushing to `main` triggers the GitHub Actions workflow that builds the site and
deploys it to GitHub Pages. To replace the hero portrait, overwrite
`public/portrait.png`.
